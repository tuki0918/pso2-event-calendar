### Requirements

+ php: >= 7.1
+ src/.env
+ src/service-account.json

### Usage

+ Calendar Update

```
$ ./run.sh deploy
```

+ Fix Coding Style

```
$ ./run.sh styling
```

+ Unit Test

```
$ ./run.sh test
```

### Composer

+ Packages Install (PHP v7.0)

```
$ ./run.sh init
```
