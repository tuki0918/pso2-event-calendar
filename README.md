### Requires

+ php: >= 7.1
+ src/.env
+ src/service-account.json

### Usage

+ Calendar Update

```
make update
```

+ Fix Coding Style

```
make styling
```

+ Unit Test

```
make test
```

### Composer

+ Packages Install (PHP v7.0)

```
make setup
```
